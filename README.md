# Программа — трекер возврата к силовой форме

Личный offline-first PWA-трекер силовых тренировок. Vanilla JS, без сборки и
зависимостей: открыл — работает. Все данные хранятся **только на устройстве**
(IndexedDB), никакого сервера и аккаунтов.

## Что умеет

- Программа A/B/C (грудь+бицепс / спина+трицепс / ноги+тяжёлый жим+плечи)
- Логирование подходов: вес, повторы, техника (чисто/читинг), боль
- Рабочие базы по упражнениям со статусами: пробный → рабочий → закреплённый
- Автопланировщик на 6 месяцев: 4 волны (вход → проработка → закрепление → пик),
  продвижение зависит от боли и читинга, прогрессия — от возраста и пола
- Замеры тела с динамикой, недельный отчёт с рекомендациями
- Текстовый отчёт о тренировке — через системный share sheet (Telegram, Заметки)
- Резервная копия: экспорт/импорт всех данных в JSON (Настройки → Данные)
- Блокировка упражнений на ноги (защита колена) до разрешения врача

## Запуск локально

```powershell
cd strength-pwa-v2
python -m http.server 8000
# открыть http://localhost:8000
```

Открытие `index.html` двойным кликом (`file://`) тоже работает — приложение
переключается на localStorage вместо IndexedDB.

## Установка на iPhone

1. Открой адрес приложения в Safari (см. «Деплой» ниже).
2. Поделиться → **На экран «Домой»**.
3. Запускай с иконки — приложение работает офлайн, интернет нужен только
   для получения обновлений.

## Деплой (GitHub Pages)

Репозиторий публичный, Pages раздаёт ветку `main` из корня. Обновление:

```powershell
git add -A
git commit -m "..."
git push
```

Через минуту-две Pages пересоберётся; телефон подхватит новую версию при
следующем запуске с интернетом (service worker работает в режиме network-first).

## Данные и бэкап — важно

- Все данные живут в браузерном хранилище устройства.
- **Удаление иконки PWA с экрана «Домой» стирает всю историю тренировок.**
- Если пользоваться сайтом в Safari без установки на экран «Домой», iOS может
  удалить хранилище после 7 дней неиспользования (ITP). Установленной PWA это
  не касается.
- Поэтому: Настройки → Данные → **Экспорт копии** — регулярно. Восстановление —
  «Импорт из файла» (полностью заменяет текущие данные).

## Структура

| Файл | Назначение |
|---|---|
| `index.html` | Оболочка, нижняя навигация |
| `app.js` | Состояние, роутер, все экраны, планировщик |
| `db.js` | Хранилище: IndexedDB + localStorage-fallback, экспорт/импорт |
| `export.js` | Share sheet / скачивание файлов / toast |
| `data/exerciseTemplates.js` | Справочник упражнений с весами и планами |
| `service-worker.js` | Офлайн-кеш, network-first обновления |
